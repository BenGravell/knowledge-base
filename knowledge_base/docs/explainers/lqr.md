# Linear Quadratic Regulator (LQR)

*Linear Quadratic Regulator (LQR)* is a feedback control scheme based on solving for the optimal control policy given quadratic stage costs and linear dynamics.

Learn more with these resources:

* [Underactuated Robotics - Ch. 8 - Linear Quadratic Regulators](https://underactuated.mit.edu/lqr.html)
* [EE363: Lecture Slides](https://web.stanford.edu/class/ee363/lectures.html)
